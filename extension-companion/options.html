ig-dl session exporter

Export your current Instagram browser session (cookies + captured API headers) as a JSON file the ig-dl CLI can import.

Before clicking: open instagram.com, make sure you are logged in, then scroll around or visit a profile so the extension can observe the rotating headers. Then come back here and export.

The downloaded file contains your Instagram session cookies. Treat it like a password — anyone with it can act as you on Instagram until the session expires.
